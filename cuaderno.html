📘 Cuaderno de Aprendizaje

1

Semana 1

Temas aprendidos: Instalación y familiarización con Visual Studio Code.

Ejercicios / Resultados: No se realizaron ejercicios específicos, pero se llevó a cabo correctamente la instalación y configuración inicial del entorno de desarrollo.

Reflexión: Esta semana fue clave para sentar las bases del curso, ya que Visual Studio Code será la herramienta que utilizaremos constantemente. Aprendí a instalarlo, explorar su interfaz y entender su importancia en el flujo de trabajo de un desarrollador. Aunque no se programó, fue un paso necesario para estar listos para las próximas semanas.

2

Semana 2

Temas aprendidos: Uso de atajos y sintaxis Emmet en HTML.

Ejercicios / Resultados: Realizamos una práctica utilizando Emmet para generar estructuras HTML de manera rápida, lo que permitió optimizar el tiempo de codificación.

Reflexión: Descubrí lo útil que es Emmet para trabajar eficientemente. Aprender estos atajos desde el inicio me ayudará a tener mayor agilidad al escribir código en HTML. Me motivó ver cómo se pueden lograr resultados complejos con tan solo unas pocas teclas.

3

Semana 3

Temas aprendidos: Exposición sobre HTML básico y avanzado.

Ejercicios / Resultados: Se presentaron conceptos clave como etiquetas semánticas, formularios, estructuras, y se discutieron ejemplos reales.

Reflexión: Esta semana reforcé mis conocimientos de HTML y comprendí mejor su estructura semántica. Las exposiciones ayudaron a repasar y organizar lo aprendido, además de exponerme a nuevas etiquetas que no conocía en profundidad.

4

Semana 4

Temas aprendidos: Exposición sobre CSS básico y avanzado, y práctica grupal.

Ejercicios / Resultados: Participamos en una práctica grupal donde aplicamos estilos con CSS, usando selectores, propiedades visuales y layouts.

Reflexión: Fue una semana muy práctica. Aprendí a trabajar en equipo y a aplicar estilos más complejos con CSS. Me ayudó a ver cómo se combinan estructura y diseño, y cómo CSS transforma una página HTML simple en algo visualmente atractivo y funcional.

5

Semana 5

Temas aprendidos: Exposición sobre Bootstrap y Tailwind CSS.

Ejercicios / Resultados: Exploramos ambas herramientas para construir interfaces rápidas y estilizadas, comparando sus ventajas.

Reflexión: Conocer Bootstrap y Tailwind me abrió los ojos a frameworks que facilitan mucho el diseño web. Me gustó la personalización de Tailwind, pero también la rapidez de Bootstrap. Ambos serán útiles dependiendo del proyecto.

6

Semana 6

Temas aprendidos: Exposición sobre JavaScript, clase práctica de React y uso de Vite.

Ejercicios / Resultados: Se comprendieron los fundamentos de JavaScript y luego se introdujo React usando Vite como entorno de desarrollo.

Reflexión: Esta semana marcó un paso importante hacia la programación funcional y de componentes. Aprender los fundamentos de JavaScript me permite interactuar con el contenido dinámico, y React me parece una herramienta muy poderosa para proyectos modernos. Vite agiliza mucho la experiencia de desarrollo.

6

Semana 7

Temas aprendidos: Uso de React y Vite

Ejercicios / Resultados: Se realizó un ejercicio guiado con los frameworks especificados

Reflexión: Se considera que el uso de Vite y React mejora el uso de los codigos de igual maner que apoya a usar un servidor dentro de los exploradores y que no dependa del mismo localhost.

🎯

Reflexión Final

Los primeros pasos realizados en el curso me apoyaron a entender concetos y funcionalidades que a pesar de usar anteriormente varios de estos, no comoncido su total funcionamiento de manera correcta
