📘 Semana 1: Entorno de desarrollo

Temas aprendidos:

  • Instalación y configuración de Visual Studio Code (VSCode).
  • Exploración básica de la interfaz: barra lateral, terminal integrada y paleta de comandos.
  • Instalación de extensiones recomendadas: Live Server, Prettier, Emmet.
  • Diferencias entre editores de texto y entornos de desarrollo.

Ejercicios / Resultados:

  • Se instaló correctamente VSCode y se personalizó con temas y fuentes.
  • Se probó la extensión Live Server creando un archivo `index.html` básico.
  • Ver ejercicio realizado

Reflexión personal:

Esta semana marcó el inicio de una experiencia de aprendizaje significativa. Pude instalar Visual Studio Code sin dificultades, y comprendí por qué es una herramienta tan popular entre los desarrolladores. Me gustó mucho la personalización que permite, así como las extensiones que facilitan el desarrollo web. El uso de Live Server para visualizar los cambios en tiempo real fue algo nuevo para mí y me resultó muy útil.

También entendí la importancia de tener un entorno de trabajo bien configurado desde el principio, ya que esto me ahorra tiempo y errores más adelante. Estoy entusiasmada por seguir aprendiendo y usar VSCode como mi herramienta principal durante el curso.
